1. O código-fonte bruto do site deve ser copiado e colado no arquivo de texto "URL.txt". Após colar, salve o arquivo.
	Obs. O tamanho total do código-fonte não pode exceder os 2000000 (2 milhões) de caracteres, senão o programa finalizará e retornará o erro 1.
	Para saber o tamanho exato de caracteres contido no seu código-fonte, utilize o site https://www.lettercount.com/ ou https://www.charactercountonline.com/

2. Após isso, inicie o programa em C. Ele irá separar todos os links com o atributo HTML href iniciados com "http" ou "https".

3. Caso todo o programa seja executado com sucesso, ele imprimirá na tela "URL's extraidas com sucesso!".

4. Todas as URL's extraídas serão armazenadas no arquivo de texto "extraidas.txt".
